### What is a Computer?

A computer is a combination of hardware and software that processes code in order to complete complex tasks.



### What is an Operating System?

An operating system is what compiles the code from the computer into something that's accessible and tangible. Usually, it displays graphics and makes the code usable.



### What is Cyber Security?

Cyber security is the process of using ethical hacking and defender techniques in order to maintain the stability and integrity of an organizations systems or code.
