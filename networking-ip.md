## IP Adresses





### What is an IP Address?



An IP Address the assigned internet protocol address for every device connected to the internet. It serves as the address used to send information over a network.



Private IP: 192.168.1.106





### Public vs. Private IP



* Private IP's are translated into Public IP's for internet use, and then translated back
* Private IP's are different for each device, while Public IP's usually stay the same
* Private IP's can often change, Private IP's are permanent and do not change



Many home devices can share one Public IP because of the ISP.
