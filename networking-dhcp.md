## DHCP Definition



DHCP automatically assigns an IP Address to a device via the DHCP Server.



## DHCP Test



Today I ran a test in the terminal. First, disconnected and reconnected from Wi-Fi. Then, I ran ipconfig /all and observed that the computer still has a valid address.



## Why is DHCP Useful?



DHCP is useful for assigning IP's automatically to new devices without having to manually set them. The DHCP Server gives the DHCP Client an IP Address from a pool of addresses.
